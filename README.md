## Description
This is a simple angular app whereby one can view user's profile on GitHub provided they have an account.It also facilitates viewing of the user's repositories.

## Setup/Installation Requirements
* Ensure you install atom text editor.
* App depends only on HTML5, CSS, Bootstrap, Angular.Js code.
* Preferrably run the app on chrome.
